jQueryOut Aufgabe

Aufgabenstellung

Wichtig: Alles mit jQuery!

1) Erzeugen Sie mit jQuery das CSS, sodass die Seite wie folgt aussieht:

  1. Mainheader – ein Abstand von 20px
  2. Die Farbe des «maintext» muss #0769AA sein
  3. Alle p-Elemente im Footer sollen fett geschrieben sein

2) Erstellen Sie im «js»-Ordner eine Datei «my.js» und speichern Sie darin den Code von der Aufgabe 1. Verlinken Sie diese Datei, sodass die Seite wie gefordert angezeigt wird.

3) Wenn auf das Bild geklickt wird, muss ein Rahmen um das Logo erstellt werden. Der Text soll vergrössert werden.

Der Text im Footer soll durch den Klick verändert werden: «Doppelklick verkleinert wieder…».

4) Erstellen Sie eine weitere Funktion mit jQuery: wenn auf das Logo ein Doppelklick gemacht wird, soll alles wieder wie vor dem Klick sein.

Lösungen

jQuery ist die meistverwendete JavaScript-Bibliothek. Jede zweite Website und drei Viertel der 10.000 meistbesuchten Websites nutzen jQuery (Stand: Juli 2014). jQuery wird in vielen Content-Management-Systemen und Webframeworks bereits mitgeliefert, zum Beispiel in Joomla, WordPress, MediaWiki oder Drupal.
