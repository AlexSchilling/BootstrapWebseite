jQuery CSS Aufgabe

1) Als Start müssen Sie ein HTML-Gerüst im «body» erstellen.

Aufgabenstellung

1) Um die Tabelle klarer zu gestalten, passen Sie das CSS an, sodass die Ausgabe in der gleichen Art wie unten aufgeführt aussieht:

Es ist wichtig, dass Sie eine eigene CSS-Datei verwenden und nicht die von Bootstrap. Andernfalls würde bei einem Update alles verloren gehen.

2) Als Nächstes ist eine Klasse nötig, um die Zellen einzufärben. Dafür gehen Sie über die Zeile «tr», erstellen Sie eine Klasse «colored» und weisen Sie diese der Zelle zu. Im Beispiel wird die Farbe «#fafad2» zugewiesen.

Jetzt könnte man bei jeder Zeile «tr» die Klasse «colored» einfügen, was aber viel zu aufwändig ist.

3) Erstellen Sie ein jQuery-Skript, das allen «tr»-Tags die Farbe zuweist.

4) Nun möchten Sie die Farbe ändern, wenn der Mauszeiger sich über einer Zelle befindet. Dafür erstellen Sie ein weiteres CSS-Format «mouseon» mit der Farbe «#1e90ff». Fügen Sie die geforderte Funktionalität mit jQuery ein.

5) Wenn eine Zeile mit der Maus eingefärbt ist, dann bleibt diese bestehen. Darum braucht es noch ein jQuery-Skript, das beim Verlassen einer Zelle die Farbe wieder zurückstellt. Implementieren Sie diese Funktionalität mit jQuery.
