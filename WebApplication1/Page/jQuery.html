Was ist jQuery?

jQuery ist eine schnelle und funktionsreiche JavaScript-Bibliothek.

  • Ist die meistverwendete JavaScript-Bibliothek
  • Wird bereits von Bootstrap verwendet

jQuery bietet JavaScript-Entwicklern folgende Funktionen:

  • Elementselektion im Document Object Model (DOM)
  • Document Object Model-Manipulation
  • Erweitertes Event Handling
  • Animationen & Effekte
  • Ajax-Funktionalitäten
  • Erweiterbarkeit durch zahlreiche freie Plug-ins
