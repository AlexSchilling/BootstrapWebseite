HTML

HTML ist eine Abkürzung für "Hypertext Markup Language". HTML ist heute der wichtigste Auszeichnungsstandard im Internet. Er ist Voraussetzung für die Programmierung und das Design von Webinhalten. Andere Standards wie PHP bauen in erheblichem Maße auf HTML auf.

Mehr Infos »

CSS

CSS bedeutet ausgeschrieben "Cascading Style Sheet" und ist eine Gestaltungs- und Formatierungssprache vor allem für Internetseiten. Sie ist Standard bei heutigem Webdesign und Layout. Mit Hilfe von CSS können nicht nur einfache Texte und Inhalte einer Webseite dargestellztund formatiert werden, sondern auch interaktive Elemente wie Navigationen aufgebaut werden.

jQuery

jQuery ist eine JavaScript-Bibliothek, die frei verfügbar ist. Diese bietet viele und einfache Funktionen zur Manipulation und Navigation von DOM (Document Object Model)-Elementen.

Video mit Bootstrap integrieren (Aufgabe 7732_Moviemaker)

Aufgabenstellung

Erstellen Sie in 2-er bis 3-er Gruppen einen Werbespot über Multimedia, der mindestens 30, maximal 60 Sekunden lang ist.

Mindestens die Hälfte des Filmmaterials muss selbst gedreht sein. Verwenden Sie nur Fremdmaterial von guter Qualität, die meisten Filme aus dem Internet haben nur ungenügende Qualität.

Der Werbespot soll mindestens 5 Schnitte enthalten, wobei mindestens 2 Überblendungen benutzt werden müssen.

Der Film soll am Anfang oder am Schluss den Filmtitel, die Namen der Filmteammitglieder, die Bezeichnung «Modul_152» und das Datum enthalten.

Vorgehen:

Machen Sie einen Plan, welche Aufnahmen Sie brauchen.

Filmen Sie die einzelnen Szenen [Thema: Filmen].

Übertragen Sie das Filmmaterial auf den PC [Thema: Film einlesen].

Schneiden und vertonen Sie den Werbespot [Thema: Schneiden, Komprimieren].

Speichern des Videos:
  1. Als Video-CD-File
  2. Komprimiert, sodass der Film über das Internet lauffähig ist.
